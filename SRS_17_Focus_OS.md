# Software Requirements Specification (SRS)
## Project 17: Focus OS — Distraction-Free Workspace (Productivity)

### 1. Project Description
Focus OS is a comprehensive productivity web application designed to help users minimize digital distractions and maintain deep work sessions. It combines Pomodoro timers, website blockers, ambient soundscapes, and goal tracking into a single, unified distraction-free environment.

### 2. Core Scope
The application relies heavily on timer logic, audio playback, and a clean, minimalist UI. Advanced features may include browser extension integration to actively block distracting websites during focus sessions.

### 3. Functionalities (Minimum 25)

#### Core Productivity Tools
1. **Pomodoro Timer:** Standard 25-minute focus / 5-minute break cycles, fully customizable.
2. **Session Types:** Options for 'Deep Work', 'Light Task', and 'Reading', each with different preset timers.
3. **Daily Goal Setting:** Users can write down their top 3 goals for the day upon logging in.
4. **Task Checklist:** A simple, integrated to-do list that links tasks to specific Pomodoro sessions.
5. **Session Pause/Resume:** Ability to temporarily halt a session without losing progress.

#### Audio & Environment
6. **Ambient Soundscapes:** Built-in audio tracks like rain, coffee shop, white noise, and lo-fi beats.
7. **Volume Mixing:** Allow users to mix multiple sounds (e.g., 50% rain + 50% lo-fi).
8. **Minimalist UI Toggle:** A button that hides all non-essential UI elements, showing only the timer.
9. **Full-Screen Mode:** Automatically enter full-screen to hide the OS taskbar and browser tabs.
10. **Visual Breath Pacer:** An optional pulsating circle to guide deep breathing during break times.

#### Anti-Distraction Features
11. **Website Blocker (via Extension):** Integrate with a companion browser extension to block URLs like YouTube or Reddit during active timers.
12. **Strict Mode:** Prevent the user from pausing or canceling the timer once it starts.
13. **Distraction Logging:** A quick input field to log "distracting thoughts" without acting on them (brain dump).
14. **Motivational Nudges:** Display random motivational quotes during breaks.
15. **Break Reminders:** Gentle notifications to stand up and stretch during 5-minute breaks.

#### Analytics & Tracking
16. **Daily Productivity Score:** A score calculated based on completed Pomodoros and checked-off tasks.
17. **Weekly Reports:** Visual charts showing which days of the week the user was most focused.
18. **Time-of-Day Heatmap:** Show when the user is statistically most productive (morning vs. evening).
19. **Streaks:** Gamified streak tracking for consecutive days of hitting the daily focus goal.
20. **Export Data:** Export focus logs as CSV or PDF for personal review.

#### User Account & Settings
21. **User Registration/Login:** Secure authentication to save data across devices.
22. **Theme Customization:** Dark mode, light mode, and color accents.
23. **Custom Alarm Sounds:** Choose different notification sounds for session end.
24. **Notification Settings:** Toggle desktop push notifications on/off.
25. **Admin Dashboard:** Monitor server uptime, user registrations, and overall app performance.
